#  Rule-Based AI Chatbot

## Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python.

The chatbot responds to predefined user inputs using if-else statements and runs continuously until the user exits.

## Features

- Greeting messages
- AI-related responses
- Continuous conversation
- Exit command
- Beginner-friendly implementation

## Technologies Used

- Python 3
- VS Code

## Project Structure

```
chatbot.py
README.md
requirements.txt
screenshots/
```

## How to Run

1. Install Python 3.
2. Download this repository.
3. Open terminal.
4. Run:

```bash
python chatbot.py
```

## Sample Output

```
You: hello
Bot: Hello! How can I help you?

You: what is ai
Bot: AI stands for Artificial Intelligence.

You: exit
Bot: Chat ended.
```
